# Abhijith Sivaprasadan Portfolio

Static GitHub Pages portfolio for energy systems modelling, grid flexibility, BESS, PyPSA, CFD/CHT and energy analytics applications.

## Local editing

Open this folder in VS Code and edit:

- `index.html` for the homepage
- `styles.css` for styling
- `projects/*.html` for individual project case studies
- `downloads/Abhijith_Sivaprasadan_CV.pdf` for the CV download

## Publish

```powershell
git add .
git commit -m "Update portfolio"
git push origin main
```

GitHub Pages URL:

```text
https://abhijith-sivaprasadan.github.io
```
