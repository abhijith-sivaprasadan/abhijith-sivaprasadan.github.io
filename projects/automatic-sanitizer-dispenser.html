Automatic Sanitizer Dispenser

A compact student prototype for contactless dispensing, included as early evidence of fast hardware iteration, sensor-actuator logic and practical build constraints.

Project snapshot

  • Context: APJ Abdul Kalam Technological University student project
  • Period: Jun 2020 to Jul 2020
  • Tools: Arduino IDE, low-cost sensing and reused components
  • Theme: contactless dispensing and rapid prototyping

Technical question

Can a simple contactless dispensing prototype be built with inexpensive electronics and reused components while keeping the sensing, actuation and refill mechanism understandable for a student team?

Method

  • Defined a compact hardware chain: proximity detection, microcontroller logic and dispenser actuation.
  • Used Arduino programming for threshold-based control rather than a complex embedded stack.
  • Prioritised available materials and e-waste reuse over custom fabrication.

What it shows

This is not a target-role centrepiece. It is useful supporting evidence because it shows comfort with practical prototyping: translating a need into a working sensor-actuator loop, making build choices under material constraints and documenting an early engineering artifact.

Limitations

The public portfolio keeps this page concise because the available evidence is limited compared with the later Siemens thesis, energy-system modelling and design-report projects.
