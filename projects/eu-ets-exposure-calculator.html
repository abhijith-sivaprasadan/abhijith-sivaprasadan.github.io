EU ETS · Carbon cost · Streamlit

EU ETS Exposure Calculator

A focused EU ETS decision-support toolkit for emissions accounting, EUA price scenarios, carbon-intensity trends and PDF reporting.

GitHub repository

Project snapshot

  • Tools: Python, Streamlit, Excel, pandas, Plotly, pytest
  • Inputs: activity data, Scope 1 factors, Scope 2/steam context, EUA price scenarios
  • Outputs: scenario summary, sensitivity curves, emissions plots and PDF export
  • Quality: workbook parsing module, CLI runner and regression tests

What I built

  • Created a Streamlit app and Excel workflow for EU ETS-focused exposure analysis.
  • Implemented calculations for direct emissions, allowance-cost scenarios and carbon intensity trends.
  • Added EUA price sensitivity views and cost-delta interpretation for management-facing decisions.
  • Packaged workbook parsing, fallback calculations, CLI execution and automated tests.

Relevance

Why this matters

This project demonstrates how regulatory carbon exposure can be converted into a usable engineering-finance tool: transparent assumptions, scenario comparison, sensitivity analysis and repeatable reporting.
