Day-Ahead Building Heating Demand Forecasting

Heating-demand forecasting using weather and temporal features, comparing baseline models and neural-network approaches.

Project snapshot

  • Tools: Python, pandas, feature engineering, regression/classification style model evaluation
  • Focus: heating demand, weather features, temporal features, model comparison
  • Type: energy-data analytics project

What I did

  • Prepared weather and time-based input features for day-ahead heating demand prediction.
  • Compared baseline methods with shallow and deeper neural-network approaches.
  • Evaluated model performance and documented assumptions, limits and modelling choices.
  • Connected forecasting output to operational energy-system decision-making.

Relevance

Why this matters

This project demonstrates time-series thinking and model-evaluation skills that are useful for energy analytics, demand modelling, BESS operation and market-data workflows.
