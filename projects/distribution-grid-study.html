KTH project · Power flow · Grid planning

Distribution Grid Study with EV, PV and Storage Impacts

Distribution-grid simulation study using a CIGRE MV-style network with 24-hour load profiles, EV charging, PV/storage integration and N-1 analysis.

GitHub repository

Project snapshot

  • Repository: PRO1_network
  • Focus: power flow, N-1 contingency, voltage stability, line loading, EV/PV/storage impacts
  • System: medium-voltage distribution network study
  • Skills: grid analysis, time-series simulation, bottleneck interpretation

What I did

  • Ran base power-flow and N-1 contingency studies to identify critical lines and loading constraints.
  • Simulated 24-hour load variation and assessed voltage/line-loading behaviour.
  • Integrated EV charging scenarios through transformer-level assumptions.
  • Analysed PV and storage impacts on losses, loading and operational strategy.

Relevance

Why this matters

This project supports grid-flow and congestion roles because it connects physical grid constraints with scenario-based planning and flexibility analysis.
