Distribution Grid Study with EV, PV and Storage Impacts

Distribution-grid simulation study using a CIGRE MV-style network with 24-hour load profiles, EV charging, PV/storage integration and N-1 analysis.

GitHub repository

Evidence dashboard

Grid flexibility framed as physical constraint analysis.

Technical question

How do EV charging, PV generation and storage alter voltage profiles, line loading and N-1 security in a medium-voltage distribution network?

24 hload variation N-1contingency EV/PVstress cases

Method

The study used base and contingency power-flow runs to identify critical loading, then added time variation, EV charging, PV and storage assumptions to test whether flexibility shifts the bottlenecks or simply moves them in time.

Outputs

The page is intentionally kept as grid-planning evidence rather than a PyPSA headline: power-flow behaviour, line loading, voltage stability and flexibility interpretation are the useful engineering outputs.

Limitations and relevance

This is coursework-scale evidence, so it should support the Energy Systems track rather than carry a standalone grid-flexibility identity. It is still useful because it connects decarbonisation technologies to the physical constraints they create in distribution networks.

Project snapshot

  • Repository: PRO1_network
  • Focus: power flow, N-1 contingency, voltage stability, line loading, EV/PV/storage impacts
  • System: medium-voltage distribution network study
  • Skills: grid analysis, time-series simulation, bottleneck interpretation

Engineering contribution

  • Ran base power-flow and N-1 contingency studies to identify critical lines and loading constraints.
  • Simulated 24-hour load variation and assessed voltage/line-loading behaviour.
  • Integrated EV charging scenarios through transformer-level assumptions.
  • Analysed PV and storage impacts on losses, loading and operational strategy.

Relevance

Why this matters

This project supports grid-flow and congestion roles because it connects physical grid constraints with scenario-based planning and flexibility analysis.
